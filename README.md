# 📊 Automated Monthly Sales Performance Dashboard | Power BI

## 📌 Project Overview

This project demonstrates the development of a fully automated multi-page Power BI dashboard designed to analyze sales performance across products, products, and regions.

The dashboard automatically updates when new monthly Excel files are added to a folder.

## 📂 Project Features

- Multi-page Power BI dashboard  
- Folder-based automated data refresh  
- KPI tracking (Sales, Profit, Orders, Margin)  
- Product-level performance analysis  
- Regional profit insights  
- Monthly trend tracking  
- Advanced visuals:
  - Decomposition Tree
  - Treemap
  - Scatter Plot
  - Heatmap Matrix

---

## ⚙️ Tools Used

- Power BI  
- Power Query  
- DAX  
- Excel  
- Folder Data Integration

---

## 🚀 Automation Feature

The dashboard automatically updates when new monthly files are added.

Example:

Sales_2025_01.xlsx  
Sales_2025_02.xlsx  
Sales_2026_01.xlsx  

After adding new files and refreshing Power BI, the dashboard updates automatically.
